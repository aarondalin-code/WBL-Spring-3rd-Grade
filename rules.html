Playoff Tie Breaker Rules
Seeding for playoffs is determined using the following tiebreakers, in order:
  1. Winning Percentage (Win%). (Wins + 0.5 √ó Ties) √∑ (Wins + Losses + Ties).
  2. Head-to-Head (two-team tie only). If exactly two teams are tied in Win%, the higher seed goes to the team that won the head-to-head matchup (only if that game did not end in a tie).
  3. Run Differential (overall). Runs For ‚àí Runs Allowed.
  4. Runs For (overall). Higher total Runs For.
  5. Runs Allowed (overall). Lower total Runs Allowed.
  6. Alphabetical Order. If still tied after all steps above, seeding is determined alphabetically by team name.
